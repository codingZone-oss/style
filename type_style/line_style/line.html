Testando o CSS em linha.

Para configurarmos estilos CSS em uma linha de um codigo HTML, usaremos de frente a tag em questão o parametro style.
Este nos ajudará a fazer todo tipo de formatações de estilos desejadas. Ex

<p style="valor do paremetro style"> Lorem ipsum dolor sit amet consectetur, adipisicing elit. Veniam repellat ducimus placeat, voluptas sint atque
beatae natus doloribus aut neque sunt quos praesentium recusandae, corrupti provident veritatis odit ipsum!
Consequatur. </p>

O CSS em linha é mais comúm em casos ligeramente simples, onde o dev interage com uma ou duas linhas
de códigos. Agora imagina que a formatação apresentada acima, tem que ser passada em mais de 10 paragrafos?
a colocarias nos restantes 10 paragrafos?

Aí como também é um caso tecnicamente simles usaresmos o CSS incorporado.

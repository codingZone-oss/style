Introdução as CSS.

Até aqui vimos como são formatadas as tags em HTML, chegou o momento de entrarmos nas formatações visuais ou
como são mundialmente conhecidas, as CSS(Casthadin Style Sheet).

Oque são as CSS

As CSS são ténicas utilizadas para as formatações de qualidade visual dos sites em html. Extremamente crucial para o
desenvolvimento WEB são resposnsáveis por embelezar os codigos HTML melhorando a interatividade garantindo a preferência
de um bom número de internautas!
